# springboot-stage
springboot脚手架

## 结构
   ----- springboot-stage         ------ 父工程\
   ----- springboot-stageapi      -------api层 对外提供服务 \
   ----- springboot-stagebiz      -------业务层\
   ----- springboot-stagedao      -------持久层\
   ----- springboot-stageservice      ---工程启动层\
